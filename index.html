CSS

CSS Selectors
Type Note
p Type Selector: Matches all elements of a given type or tag name
.calendar Class selector: Matches elements based on their class
#first-item ID selector: Must be unique and used to style only a single element
h1, h2 Group of CSS Selectors: Match multiple selectors to the same CSS rule, using a comma
h3.section-heading Chaining Selectors: Elements muct match all specified selectors
div p Descendant Selector: Match elements that are descended from another

CSS Properties
Property Note
background-color Controls the background color of elements.
!important Used on declarations to override any other declarations for a property and ignore selector specificity.
opacity Ranges from 0 (transparent) to 1 (opaque).
